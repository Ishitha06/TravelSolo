# Travel Solo - AI Powered Solo Travel Planner

Whenever we think about planning a solo trip, there are so many questions in our mind and the stress of researching about places, hotels/restaurants, safety etc.
Even though solo traveling has become the trend these days, yet this change is not consistent with the ways with which we are planning our trips.

Still one has to :
- Search multiple websites
- Compare hotels/restaurants
- Estimate budget
- Customizing it based on their vibe and preferences.

We sensed the need for a platform where users can smartly plan their itinerary just by answering simple questions about their interests and preferences.
Hence, Travel Solo.

Travel Solo is an AI powered website that generates personalized travel itinerary for the user.
It takes user responses about:
- Destination
- Duration
- Budget
- Travel pace
- Vibe
- Food preference

It also:
- Suggests attractions
- Recommends hotels
- Estimates daily budget
- Includes currency conversion

# Tech stack used is 

- For Frontend: HTML,CSS,Javascript
- For Backend: Node.js, Express.js
- API's : Open Trip map (tourist attraction data ) and Open router (AI itinerary generation)

##  Features

- AI-generated day-wise itinerary
- Morning, Afternoon, Evening planning
- Budget estimation
- Hotel suggestions
- Currency converter
- Fully responsive UI
